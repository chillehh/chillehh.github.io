Island Generation

Project information

  • Project start date: Aug, 2022
  • Project URL: GitHub Link
  • Skills: C#, Unity 3D

Iso Island is a small project focused on island generation in an isometric context. The algorithm uses Perlin Noise to generate an island with it's layers of elements. The algorithm also accomplishes the placement of 4 houses in opposite corners of the island after generation of said island and is a way to show how a game could be created by placing objects over the island in particular orders to keep a game's rules and flow in tact.

Procedural generation is something that amazes me as I really enjoy how such a simple algorithm can generate such wide varied and complex visuals. I've done work in procedural generation before, but not with Perlin Noise, so this was a fun experience to try to mimic landmasses closer to the real world.

The last image features the original hex grid algorithm output that this is project is based on. I created that a few months prior as a project with a main focus of teaching myself multiplayer networking in Unity3D.
